# 文档管理系统

一个完整的文档管理系统，支持文档上传、下载、分类管理、用户权限控制等功能。

## 系统架构

- **前端**：Vue 3 + Vite + Element Plus
- **后端**：Flask + SQLAlchemy + JWT
- **数据库**：MySQL
- **文件存储**：本地文件系统 + FTP（可选）

## 系统功能

### 核心功能
- 用户认证系统（登录/注册/JWT授权）
- 文档管理（上传、下载、删除、搜索）
- 分类管理（多级分类、文档归类）
- 用户权限管理（角色权限控制）
- 文档预览（支持常见文档格式）

### 管理员功能
- 用户管理（增删改查用户信息）
- 系统配置管理
- 操作日志查看


## 技术特点

1. **前后端分离架构**：清晰的职责划分，便于维护和扩展
2. **JWT认证机制**：安全的用户认证和权限控制
3. **响应式设计**：适配不同设备屏幕
4. **模块化开发**：代码结构清晰，易于扩展
5. **完整的错误处理**：友好的用户提示和日志记录

## 开发说明

### 后端开发

- API遵循RESTful设计规范
- 使用SQLAlchemy进行数据库操作
- JWT用于身份验证和授权

### 前端开发

- 基于Vue 3 Composition API
- 使用Pinia进行状态管理
- Vue Router处理路由导航
- Element Plus提供UI组件

## 注意事项

1. 确保前后端服务都正常运行
2. 检查API地址配置是否正确
3. 首次使用需要创建管理员账号
4. 生产环境部署时请修改JWT密钥等敏感配置
